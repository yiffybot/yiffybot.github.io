How does Yiffy work?

We want Yiffy to be as transparent as possible - so here's how it works!
Images
We use an image database!

Yiffy uses a database filled with thousands of discord image links - all of furry porn collected by us.

Command
We use just one command!

Our one command, y!yiff, tells Yiffy to display a random image from our database as an embed.

(You have no idea how long it took to find a relatively SFW image...)

Permissions
We automatically set ourselves!

Some users questions Yiffy's need for administrator permissions - don't worry! This is so our bot can automatically set itself up immediately, thus removing any complicated configuration!
